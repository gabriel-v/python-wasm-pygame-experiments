Downloads

To install Thorpy, just type pip3 install thorpy in a console.

ThorPy installers for windows, built and source are uploaded on the PyPI stack. Releases can be found on https://pypi.python.org/pypi/thorpy.

The source code and a list of modifications is also published on GitHub at https://github.com/YannThorimbert/Thorpy.

Note that an installation tutorial is avalaible.
