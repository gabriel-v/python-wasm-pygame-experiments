Tutorials - Installation tutorial

This tutorial is for people who already installed Python and pygame. If not, please install Python 2.7 or any version of Python 3 (Python download page for beginners) and the corresponding pygame release. If you are completely new with Python, you can get started here.

ThorPy installers for windows, built and source are uploaded on the PyPI stack ; its releases can be found on https://pypi.python.org/pypi/thorpy/.

The source code and a list of modifications is also published on GitHub at https://github.com/YannThorimbert/Thorpy.

We give here three different installation methods for Windows OS, one for Linux OS and one way to use ThorPy without installing it on your machine, which works for all systems.

Windows

Install with installer

  1. On https://pypi.python.org/pypi/thorpy, download thorpy-1.5.5b0.win-amd64.exe if you need the 64-bit version (works on 64-bit OS with 64-bit Python version), or thorpy-1.5.5b0.win32.exe if not.
  2. Run the installation file and follow instructions. If the installer cannot Python in register, you are probably trying to install the wrong version of ThorPy (64-bit instead of 32, or 32 instead of 64).
  3. Once the installation is completed, you can open a Python shell and test if ThorPy was successfully installed by typing the following instruction: from thorpy import testmodule; testmodule.run(). If a window with some buttons appears, then ThorPy is working.

Manually install from source

Since it is written in pure Python, ThorPy is really easy to install manually from source.
  1. On https://pypi.python.org/pypi/thorpy/, download thorpy-1.5.5b0.zip.
  2. Go to your Python directory, in Lib\site-packages folder (for example, C:\Python34\Lib\site-packages\). Extract the thorpy folder. That's all.
  3. You can open a Python shell and test if ThorPy was successfully installed by typing the following instruction: from thorpy import testmodule; testmodule.run(). If a window with some buttons appears, then ThorPy is working.

Build and Source from PyPI

Build and source files for windows are uploaded on PyPI (https://pypi.python.org/pypi/thorpy/). Thus, you can use traditional pip install commands if you are familiar with this method.

Linux

Manually install from source

Since it is written in pure Python, ThorPy is really easy to install manually from source. For the moment, this is the only possible installation for Linux systems.
  1. On https://pypi.python.org/pypi/thorpy/, download thorpy-1.5.5b0.tar.gz or thorpy-1.5.5b0.zip.
  2. Go to your Python directory, in site-packages folder (for example, /usr/local/lib/python3.4/site-packages/). Extract the thorpy folder. That's all.
  3. You can open a Python shell and test if ThorPy was successfully installed by typing the following instruction: from thorpy import testmodule; testmodule.run(). If a window with some buttons appears, then ThorPy is working.

All Plateforms

Use ThorPy without installing it

You can use ThorPy without installing it. However, this kind of use is less general than the other ones. We advise to properly install ThorPy if possible.
  1. On https://pypi.python.org/pypi/thorpy/, download thorpy-1.5.5b0.tar.gz or thorpy-1.5.5b0.zip.
  2. Extract the thorpy folder anywhere you want in your computer.
  3. All the scripts which are located in the same folder as thorpy can use ThorPy as if it was installed. Note that this is a way to distribute your scripts.

(optional) Installing dependencies

ThorPy has no other critical dependencies than pygame. However, some graphical effects like shadows and blurs need NumPy and PIL (Python Imaging Library) to work properly. If you do not have these libraries, you can use ThorPy, but without these graphical effects (by the way, see graphics tutorial if you want to make your code safe for people who don't have PIL and NumPy on their computer). You can check if ThorPy is able to produce shadows by checking the value of HAS_NUMPY, HAS_PIL and HAS_SHADOW in the constants module. For example: if thorpy.constants.CAN_SHADOWS: thorpy.makeup.add_button_shadow(my_button).

NumPy can be found on http://www.numpy.org, PIL for python 2.7 on http://www.pythonware.com/products/pil/ and for python 3.x we advise to use Pillow, which provides installation instructions on http://pillow.readthedocs.org/installation.html .
