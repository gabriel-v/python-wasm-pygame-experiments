Examples - Animated gif

Full code

The code below shows how to set up an animated gif.


        import thorpy

        application = thorpy.Application((800, 600), "Example of animated gif")

        gif_element = thorpy.AnimatedGif("../documentation/examples/myGif.gif")
        gif_element.center()

        menu = thorpy.Menu(gif_element)
        menu.play()

        application.quit()

        """
        Other parameters to pass to the constructor of AnimatedGif:
                : the path to the image.
                : if path is None, use this color instead of image.
                : increase this parameter to lower the gif speed.
                : number of times the gif is played
        """
